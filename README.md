# STUDY E-commerce

## Overview

It is a study e-commerce project built with React, Mobx, and Framer Motion. It includes features such as a shopping cart, animations powered by Framer, and utilizes Mobx for state management. The project also interacts with a RESTful API to fetch and display product data.

![Example Screenshot](https://i.imgur.com/RHPu0eU.png)

## Features

- **Shopping Cart:** Utilizes local storage to persist the shopping cart across sessions.
- **Animations:** Framer Motion is used for smooth and engaging UI animations.
- **REST API:** Interacts with a RESTful API to fetch and display product information.
- **Skeleton Loading:** Provides a seamless user experience with skeleton loading while content is being fetched.

## Technologies Used

- React
- Mobx
- Framer Motion
- TypeScript
- Sass Modules
